## Prerequisites

<Callout title="You need one target environment">
  Use a deployed Artifacts instance, or run the local monorepo stack with the web
  app on `localhost:3000` and the API on `127.0.0.1:3001`.
</Callout>

<Cards>
  <Card title="Runtime" description="Bun for local development and CLI builds." />
  <Card title="Identity" description="Browser login for humans, bearer tokens for CI and agents." />
  <Card title="Project" description="A project slug where new artifacts should be published." />
</Cards>

## Publish in five steps

<Steps>
  <Step>
    ### Install the CLI

    <Tabs items={["Published", "Monorepo"]}>
      <Tab>
        ```bash
        npx @agent-artifacts/cli@latest setup
        ```
      </Tab>
      <Tab>
        ```bash
        bun install
        bun run cli:build
        bun run cli:install
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step>
    ### Point the CLI at an environment

    <Tabs items={["Local", "Production"]}>
      <Tab>
        ```bash
        export AGENT_ARTIFACTS_BASE_URL="http://127.0.0.1:3001"
        export AGENT_ARTIFACTS_WEB_URL="http://localhost:3000"
        ```
      </Tab>
      <Tab>
        Production bundles can bake default API and web URLs into the binary,
        but flags and environment variables still win at runtime.
      </Tab>
    </Tabs>
  </Step>

  <Step>
    ### Authenticate

    <Tabs items={["Browser login", "Automation token"]}>
      <Tab>
        ```bash
        artifacts login
        artifacts whoami
        ```
      </Tab>
      <Tab>
        ```bash
        export AGENT_ARTIFACTS_TOKEN="..."
        export AGENT_ARTIFACTS_NO_INPUT=1
        artifacts whoami --format json
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step>
    ### Publish a file

    ```bash
    artifacts push --owner alice --project-slug default --file ./report.md
    ```

    `push` infers artifact type, title, and slug from `.md`, `.markdown`,
    `.html`, `.htm`, `.jsx`, and `.tsx` files.
  </Step>

  <Step>
    ### Update the same artifact

    ```bash
    artifacts push --owner alice --project-slug default --file ./report.md --slug report
    ```

    Updates append immutable versions. The artifact URL stays stable while
    history, diffs, and restore actions remain available.
  </Step>
</Steps>
