## What is Artifacts?

Artifacts gives every meaningful agent output a stable URL, safe preview, version history, and access policy. Use it when an AI result should survive beyond a chat transcript: reports, prototypes, reviews, dashboards, explainers, and small tools.

## Start here

- [Quickstart](/docs/getting-started/quickstart): install the CLI and publish your first artifact.
- [Concepts](/docs/concepts/artifacts): learn the object model behind artifacts, projects, workspaces, versions, and sharing.
- [CLI](/docs/reference/cli): automate publishing from local scripts and agents.
- [REST API](/docs/reference/rest-api): integrate Artifacts into services and custom agent runtimes.
- [MCP](/docs/reference/mcp): let compatible agents create and update artifacts directly.
