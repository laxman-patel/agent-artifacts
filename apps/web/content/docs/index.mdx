## What is Artifacts?

<Callout type="idea" title="Docs for agent-generated work that should last">
  Artifacts gives every meaningful agent output a stable URL, safe preview,
  version history, and access policy. Use it when an AI result should survive
  beyond a chat transcript: reports, prototypes, reviews, dashboards,
  explainers, and small tools.
</Callout>

## What gets stored

<Tabs items={["Markdown", "HTML", "JSX/TSX"]}>
  <Tab>
    Specs, reports, research notes, implementation plans, PR summaries, and
    other text-first artifacts.
  </Tab>
  <Tab>
    Standalone reports, visual explainers, dashboards, and generated pages that
    need safe previewing.
  </Tab>
  <Tab>
    Interactive Preact-backed prototypes, throwaway editors, and one-use tools
    created during agent workflows.
