## Global Conventions

- Inputs are named flags; there are no positional arguments.
- `--no-input` disables prompts.
- `--dry-run` previews supported mutations.
- `--format json` forces JSON output in interactive terminals.
- `--ndjson` streams list output one record per line.

## Authentication

```bash
artifacts login
artifacts logout
artifacts whoami
```

## Discovery

```bash
artifacts schema
```

Use this command for agent integration. It returns command definitions, schemas, examples, and output contracts.

## Projects

```bash
artifacts project list
artifacts project create --json '{"ownerUsername":"alice","slug":"default","title":"Default"}'
artifacts path project --owner alice --project-slug default
```

## Artifacts

```bash
artifacts push --owner alice --project-slug default --file ./report.md
artifacts artifact list --limit 20
artifacts artifact create --json '{"...":"..."}'
artifacts artifact get --artifact-id ARTIFACT_ID
artifacts artifact content --artifact-id ARTIFACT_ID
artifacts artifact update --artifact-id ARTIFACT_ID --json '{"content":"# v2"}'
artifacts artifact delete --artifact-id ARTIFACT_ID --dry-run
artifacts path artifact --owner alice --project-slug default --slug report
```

## Sharing and Audit

```bash
artifacts share create --artifact-id ARTIFACT_ID --json '{"role":"viewer"}'
artifacts share revoke --share-link-id SHARE_LINK_ID
artifacts audit list --artifact-id ARTIFACT_ID --all
```

## Exit Codes

- `0`: success.
- `2`: invalid request.
- `3`: not found.
- `4`: forbidden or unauthenticated.
- `5`: conflict.
- `69`: retryable network failure.
