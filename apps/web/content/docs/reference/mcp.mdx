## Connection

<Callout type="info" title="Connect through the web origin locally">
  `POST /mcp` is the MCP endpoint. Local clients should usually connect through
  the web app origin so rewrites and OAuth metadata match the browser-visible URL.
</Callout>

## Tool Groups

<Tabs items={["Principal", "Projects", "Artifacts", "Sharing", "Workspace"]}>
  <Tab>
    ```txt
    get_current_principal
    ```

    Returns the authenticated principal for the request.
  </Tab>
  <Tab>
    ```txt
    check_project_slug_availability
    create_project
    list_projects
    ```

    Prepare a namespace before publishing artifacts.
  </Tab>
  <Tab>
    ```txt
    check_slug_availability
    create_artifact
    update_artifact
    restore_artifact_version
    resolve_path
    ```

    Create writes the first immutable version. Update appends the next one.
  </Tab>
  <Tab>
    ```txt
    create_share_link
    list_share_links
    revoke_share_link
    ```

    Share-link tools enforce artifact permissions and account entitlements.
  </Tab>
  <Tab>
    ```txt
    list_workspaces
    list_workspace_artifacts
    list_audit_events
    ```

    Discover team context and inspect change history.
  </Tab>
</Tabs>
