## Dashboard

The dashboard is the human control surface for artifacts. Use it to browse personal and workspace artifacts, create artifacts, switch workspaces, and open project-scoped views.

## Artifact Viewer

The artifact URL renders the current version of an artifact. Supported content is displayed through controlled viewers:

- Markdown is rendered through a sanitized Markdown pipeline.
- HTML is previewed with an iframe and a content security policy.
- JSX/TSX is rendered as a Preact-backed interactive surface.

## History and Diff

Artifact subpages expose version history and version-to-version diffs. Use them to inspect changes from agents, compare generated revisions, and decide whether to keep or restore a version.

## Settings and Access

Artifact settings expose access policy controls, share-link management, and destructive actions. Workspace and account settings expose billing, API keys, audit history, and membership controls.

## Marketing Surfaces

The landing and pricing pages use the same brand tokens as docs: dark background, crisp borders, Geist fonts, and the orange artifact accent.
